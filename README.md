## 📖 Descripción

Este proyecto consiste en el desarrollo de tres simulaciones utilizando el elemento `<canvas>` de HTML5 y JavaScript, con el objetivo de representar diferentes comportamientos de partículas (círculos):

- 🔵 Movimiento libre  
- 🟡 Rebote en los bordes  
- 🔴 Detección de colisiones  

Cada simulación se ejecuta de manera independiente en su propio canvas dentro de la misma página web.

---

## ⚙️ Funcionalidades

✔ Animación en tiempo real con `requestAnimationFrame`  
✔ Generación de múltiples partículas  
✔ Detección de colisiones entre círculos  
✔ Cambio de color al colisionar  
✔ Rebote contra los límites del canvas  
✔ Control dinámico de velocidad mediante slider  
✔ Interacción con el usuario (click para agregar partículas)  
✔ Diseño visual moderno (navbar, footer, efectos visuales)

---

---

## 🧠 Tecnologías utilizadas

- HTML5  
- CSS3

---

## 🚀 Uso

1. Abrir el archivo `index.html` en el navegador  
2. Visualizar las tres simulaciones:
   - Rebote
   - Colisiones
   - Movimiento
3. Ajustar la velocidad con el control deslizante  
4. Hacer clic sobre cualquier canvas para agregar nuevas partículas  

---

## 🎨 Características visuales

- Interfaz con diseño moderno  
- Efectos de iluminación (glow) en partículas  
- Fondo degradado  
- Organización en contenedores responsivos  

---

## 📌 Notas

- Cada simulación funciona de manera independiente  
- El sistema de colisiones se basa en la distancia entre centros  
- La velocidad es controlada globalmente mediante un multiplicador

## 🏁 Conclusión

Este proyecto permite comprender conceptos fundamentales de animación, interacción y detección de colisiones en gráficos por computadora, aplicando principios básicos de física y programación en JavaScript.
